# 🇹🇼 台灣政治事件紀錄

一個開放的政治事件時間軸網站，任何人都可以新增、查看台灣的政治事件。

## 🌐 網站連結

**正式網站：** https://nillcheng.github.io/political-events-tw/

## ✨ 功能特色

- 📅 時間軸式呈現所有政治事件
- ➕ 任何人都可以透過 GitHub Issue 新增事件
- 🔍 即時搜尋功能
- 📱 響應式設計，手機也能用
- 🆓 完全免費，資料公開透明

## 🚀 如何新增事件

### 方法一：透過網站（推薦）

1. 前往網站：https://nillcheng.github.io/political-events-tw/
2. 點擊「➕ 新增事件」按鈕
3. 填寫事件資訊
4. 提交即可

### 方法二：直接在 GitHub 開 Issue

1. 點擊 [New Issue](https://github.com/nillcheng/political-events-tw/issues/new?template=event.md)
2. 選擇「新增政治事件」範本
3. 填寫以下資訊：
   ```
   日期：2026 年 5 月 8 日
   事件：你的事件描述
   來源：新聞連結（選填）
   ```
4. 提交 Issue

## 📋 事件格式

每個事件需要包含：

| 欄位 | 必填 | 說明 |
|------|------|------|
| 日期 | ✅ | 年份和月份必填，日期選填 |
| 事件 | ✅ | 簡短描述事件（建議一行文字） |
| 來源 | ❌ | 新聞連結或資料來源 |

**範例：**
```
日期：2026 年 5 月 8 日
事件：蔣萬安老鼠事件
來源：https://example.com/news
```

## 🛠️ 技術架構

- **前端：** HTML + CSS + JavaScript
- **資料儲存：** GitHub Issues
- **部署：** GitHub Pages
- **API：** GitHub REST API

## 📜 注意事項

1. **客觀中立**：請客觀描述事件，避免情緒性或攻擊性用詞
2. **一事一 Issue**：每個 Issue 只記錄一個事件
3. **資料正確**：請確認日期和事件描述正確
4. **尊重他人**：禁止仇恨言論或人身攻擊
